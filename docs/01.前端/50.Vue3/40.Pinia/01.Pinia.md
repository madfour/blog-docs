### Pinia优点
- 1.Vue3、Vue2都支持
- 2.抛弃了Mutations的操作，只有state、getters和actions
- 3.不需要嵌套模块，符合Vue3的Composition api
- 4.完整的TypeScript支持
- 5.代码更加简洁

### 安装

```bash
yarn add pinia
# or npm
npm install pinia
```
