**谁调用的，`this` 就指向谁，默认情况下指向全局对象(浏览器环境是`windows`，在 nodejs 中指向这个文件的 `module` 对象)**
`this`是一个关键字，它代表函数运行时，自动生成的一个内部对象，只能在函数内部使用。
